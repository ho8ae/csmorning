### 🤔 만들게 된 이유

코딩테스트, 프로젝트, CS 인터뷰 등 학습량이 너무 많아서 하나라도 *자동화*를 하고 싶었습니다.

**오픈소스**나 **여러 커뮤니티**를 통해 도움을 받고 있는 입장이라, 마찬가지로 **개발자나 취준생들을 위한 서비스**를 만들어 기여를 하고 나누고 싶었습니다.

### 카카오 싱크플러그인 도입
채널 알림톡을 발송을 하기 위해서는 전화번호의 입력이 필필요합니다.  `폼 작성` 이나 `사용자의 정보를 일일이 기입` 하는 경우 사용자는 *피로감*을  느낀다고 생각했습니다.

따라서 채널 내에서 `간편 회원가입` 을 할 수 있게 도입하였습니다.

<img src="https://velog.velcdn.com/images/ho8ae/post/e37e772e-32e1-41cd-860d-07e11e91cbcc/image.png">

이전에 기록했던 답변들이 사라지는 걸 막기 위해 `계정 연동 기능`을 구현하여 연동 전에 기록들이 삭제되는 것을 방지하였습니다.
