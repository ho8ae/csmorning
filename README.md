## 📝 프로젝트 소개
![alt text](<스레드 포스트.png>)
CS Morning은 개발자들이 매일 아침 CS(Computer Science) 관련 질문을 받아볼 수 있는 서비스입니다. 기술 면접 준비나 CS 지식 향상에 도움을 드립니다.

## ✨ 주요 기능

### 🎯 데일리 모드
- 매일 아침 8시 새로운 CS 질문 제공
- 즉시 답변 입력 및 정답 확인
- 운영체제, 네트워크, 데이터베이스 등 다양한 CS 분야 커버

### 📚 위클리 모드
- 매일 새로운 CS 학습 컨텐츠 제공
- 일주일 동안 학습한 내용 기반의 주간 테스트
- 매주 일요일 7문제 순차 해결 방식

### 💫 공통 기능
- 학습 모드 선택 가능 (데일리/위클리)
- 모바일 친화적인 반응형 웹 디자인
- 카카오톡 채널 알림톡 서비스

## 🚀 시작하기

1. [csmorning.co.kr](https://csmorning.co.kr)에 접속합니다.
2. 채널 추가 후 카카오 싱크를 통한 채널 내 회원가입한다.
3. 매일 오전 8시에 제공되는 새로운 질문을 확인한다.

## 🛠 기술 스택

- Frontend: React.js
- Backend: Node.js
- Database: postgresql
- Deployment: AWS
